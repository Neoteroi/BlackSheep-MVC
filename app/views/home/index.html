About

BlackSheep is a fast web framework for Python asyncio, inspired by Flask and ASP.NET Core. Its unique features, among web frameworks for Python, are built-in support for dependency injection, and a flexible model to define request handlers.

BlackSheep adheres to ASGI: it must be used with an HTTP server like uvicorn, daphne, or hypercorn.
BlackSheep targets the most recent versions of Python language.
