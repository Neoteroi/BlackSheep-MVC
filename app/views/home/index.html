Getting started

  1. Read the tutorial
  2. Familiarize with the folder structure of the project
  3. Learn how to define request handlers
  4. Familiarize with DI and model binding
  5. Refer to the full documentation
            
from dataclasses import dataclass
from blacksheep.server.controllers import Controller, get, post, delete
from blacksheep.server.bindings import FromJson


@dataclass
class Example:
    name: str
    foo: str


class Home(Controller):

    @get()
    def index(self):
        return self.view()

    @post("/item")
    def create_item(self, item: FromJson[Example]):
        ...

    @delete("/item/:item_id")
    def delete_item(self, item_id: str):
        ...

Use the provided strategy to defined API endpoints:

            
from blacksheep import Response
from blacksheep.server.controllers import ApiController, delete, get, patch, post


# In this case, the entity name is obtained from the class name: "cats"
# To specify a @classmethod called "class_name" and returning a string, like in the
# Foo example below.
class Cats(ApiController):
    @get(":cat_id")
    def get_cat(self, cat_id: str) -> Response:
        """
        Handles GET /api/cats/:id
        """

    @patch(":cat_id")
    def update_cat(self, cat_id: str) -> Response:
        """
        Handles PATCH /api/cats/:id
        """

    @post()
    def create_cat(self) -> Response:
        """
        Handles POST /api/cats
        """

    @delete(":cat_id")
    def delete_cat(self, cat_id: str) -> Response:
        """
        Handles DELETE /api/cats/:id
        """


class FooExample(ApiController):
    @classmethod
    def class_name(cls) -> str:
        return "foo"

    @get(":cat_id")
    def get_cat(self, cat_id: str) -> Response:
        """
        Handles GET /api/foo/:id
        """

About

BlackSheep is a fast web framework for Python asyncio, inspired by Flask and ASP.NET Core. Its unique features, among web frameworks for Python, are built-in support for dependency injection, and a flexible model to define request handlers.

BlackSheep adheres to ASGI: it must be used with an HTTP server like uvicorn, daphne, or hypercorn.
BlackSheep targets the most recent versions of Python language.
